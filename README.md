
# 🛡️ CodeViruss

A modern cybersecurity web application developed during my internship using React, Vite, and Tailwind CSS.  
This project focuses on creating a fast, responsive, and user-friendly cybersecurity platform with modern UI components and optimized frontend performance.

---

# 🚀 Features

- Modern responsive UI
- Cybersecurity themed website
- Built with React + Vite
- Tailwind CSS styling
- Fast page rendering
- Mobile-friendly design
- Reusable components
- Service section
- About section
- Contact section
- Smooth navigation

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Vite
- Tailwind CSS
- JavaScript
- HTML5
- CSS3

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/your-username/codeviruss.git
```

```bash
cd codeviruss
```

---

# 📦 Install Dependencies

```bash
npm install
```

---

# ▶️ Run the Project

```bash
npm run dev
```

Application runs on:

```bash
http://localhost:5173
```

---

# 🏗️ Build for Production

```bash
npm run build
```
